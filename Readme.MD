SpacyMote!
==========

Created for the DevICT Game Demo Day April 9th, 2016.

Based on the excellent example at https://github.com/xicombd/phaser-multiplayer-game

# Getting started

```
npm install
npm start dev
```

Navigate to http://localhost:3000/
